# 01-standards — Stable Reference

Coding standards, naming conventions, architecture principles, and system landscape documentation. Changes rarely; pages here are `evergreen`.

| Subfolder       | Holds                                            | Naming                     |
| --------------- | ------------------------------------------------ | -------------------------- |
| `coding/`       | ABAP coding and naming standards                 | `Standard - {Name}.md`     |
| `architecture/` | Architecture principles and documented tradeoffs | `Architecture - {Name}.md` |
| `landscape/`    | Systems, clients, transport routes               | `Landscape - {Name}.md`    |

Each subfolder contains its `_Template-*.md` — copy it when creating a new page.
