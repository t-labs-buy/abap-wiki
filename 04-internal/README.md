# 04-internal — Team Operations

Internal-only: how the team runs. Not client-facing.

| Subfolder     | Holds                                                                           | Naming                    |
| ------------- | ------------------------------------------------------------------------------- | ------------------------- |
| `contacts/`   | POC directory: who owns what workstream, module, or system                      | `POC - {Scope}.md`        |
| `onboarding/` | How a new team member gets productive                                           | `Onboarding - {Topic}.md` |
| `processes/`  | Step-by-step team processes with quality gates                                  | `Process - {Name}.md`     |
| `runbooks/`   | Operational steps for recurring moments (transport release, cutover, hypercare) | `Runbook - {Name}.md`     |
